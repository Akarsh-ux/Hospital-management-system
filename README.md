# 🏥 Hospital Management System

## 📖 Overview
A web-based Hospital Management System built using Flask, Python, HTML, CSS, JavaScript, and SQLite.

## ✨ Features
- 🔐 User Login
- 👨‍⚕️ Doctor Management
- 🧑‍🤝‍🧑 Patient Management
- 📅 Appointment Booking
- 💳 Billing
- 📊 Dashboard
- 📝 Audit Logs

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Git

## 🚀 Installation

```bash
pip install -r requirements.txt
python app.py
